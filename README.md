# سیستم خرید و فروش طلا (Gold Trading System)

## مهم
**داکیومنت کامل پروژه در فایل [Documentation.pdf](./Documentation.pdf) در روت پروژه قرار دارد.**

**فایل Collection مربوط به Postman با نام [Gold_Trading_API.postman_collection.json](./Gold_Trading_API.postman_collection.json) در روت پروژه موجود است.**

## معرفی
این پروژه یک سیستم API-Based برای خرید و فروش طلا است که با Laravel و MySQL پیاده‌سازی شده است. کاربران می‌توانند درخواست‌های خرید یا فروش طلا را با قیمت مشخص ثبت کنند و سیستم به صورت خودکار، سفارش‌های همخوان را مطابق کرده و معاملات را انجام می‌دهد.
